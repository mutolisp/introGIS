5  介面與使用基礎

5.1 安裝 QGIS

5.1.1 下載 QGIS

請至 QGIS 官方網頁下載。 網站會自動偵測你的作業系統，並提供下載的連結(Figure 5.1)

以 Windows 作業系統為例，當點選「立即下載」後，下載頁面會轉到 windows 專屬的下載頁面(Figure 5.2)。 預設會下載獨立安裝的套件(standalone)，對於新手而言可以選這個最新的版本。 以 Figure 5.2 為例，即 3.28 版，或是另外一個比較舊的長期維護釋出版(Long-Term Release, LTR)， 目前是 3.22-LTR。如果你有特別的安裝需求，像是需要安裝或不安裝特定的選項， 可以使用 OSGeo4W Network Installer 來透過網頁下載並安裝。

Note

如果沒有特別需求的話，可以直接下載穩定版(3.28)安裝即可。

5.2 QGIS 圖形介面介紹

QGIS 的預設介面可以分為四個部分：工具列(toolbars，Figure 5.3 中的1)、繪圖區(canvas，即 Figure 5.3 中的2)、 面板(panels，Figure 5.3 1中的3、4和5)與狀態列(圖1中的6)。QGIS 使用 Qt 的圖形介面框架， 因此能夠將工具列和繪圖區任意移動排列，甚至變成浮動的子視窗。要開啟自定介面，包括啟用或關閉工具列、 面板等，只要在工具列上任一位置按滑鼠右鍵就能開啟設定選單(Figure 5.4)。在選單中勾選要啟用或關閉的工具列或面板即可。 另外一個方式就是在選單中的檢視(view)點選工具列(toolbars)或面板(panels)中的選項。

5.2.1 工具列(toolbars)

在 QGIS 中的工具列如 Figure 5.6 所顯示。在最左方(水平排列下)或最上方(垂直排列下)會有個兩排的點， 我們可以稱為移動工作移動工具列的「把手」，你可以用滑鼠左鍵點這個部分拖曳到你想要的地方( 可參考 Figure 5.7)。 另外若每個工具列圖示(icon)右方有個向下的箭頭，代表這個功能圖示有子項目， 點選箭頭後會將子項目選單開啟，此時就能直接選取子項目之功能圖示。 因工具列可能因為載入圖示項目較多而自動隱藏，若有自動隱藏的工具列功能圖示時， 在該工具列的右方會有「》」的圖示，按該圖示後就會把隱藏的功能圖示都顯示出來。

QGIS 發展到現在，功能十分豐富，每次更新都會加入許多新功能，對於新手來說可能會難以切入， 因此我們把工具列整理於 Table 5.1 中，並在後方詳細說明不同工具列所提供的功能和簡單範例。 如果不清楚圖示的功能是什麼，可以將滑鼠停留在圖示上方稍待半秒左右，QGIS 就會自動出現該圖示的功能說明和快速鍵。

5.2.1.1 專案工具列(projects)

專案工具列如圖 Figure 5.8 所顯示，具有六個主要功能， 和建立與管理專案有關。詳細的說明請參見 Chapter 6 章節。 以下將簡要介紹專案工具列的元件說明，請同時參考 Figure 5.8 的編號來閱讀。

  • P1: 建立新專案(快速鍵 Ctrl + N 或 Command + N [mac])，建立新的 QGIS 專案
  • P2: 開啟專案(快速鍵 Ctrl + O 或 Command + O [mac])，開啟 QGIS 專案檔
  • P3: 儲存專案(快速鍵 Ctrl + S 或 Command + S [mac])，儲存 QGIS 專案檔
  • P4: 新建列印版面配置，建立列印出圖的版面配置
  • P5: 顯示版面配置管理，顯示版面配置的管理視窗
  • P6: 管理樣式，主要是管理顯示向量圖徵之樣式，包含顏色、符號等

5.2.1.2 地圖導覽工具列(map navigation)

  • N1: 移動地圖位置(pan map)
  • N2: 移動地圖位置到所選擇的圖徵中，會將所選的圖徵置放到繪圖區的正中央
  • N3: 地圖放大(zoom out)
  • N4: 地圖縮小(zoom out)
  • N5: 縮放至全覽圖(zoom full)，縮放到所有可見圖層的範圍
  • N6: 移動並放大所選擇的圖徵並置放在繪圖區正中央，和 N2 不同的是此功能會放大地圖
  • N7: 縮放至該圖層的範圍(extent)
  • N8: 縮放地圖至 1:1 的大小
  • N9: 縮放至上次可見的範圍
  • N10: 縮放至下一個可見之範圍
  • N11: 新開啟一個 2D 的地圖檢視(map view)，也就是會開啟一個浮動的子視窗顯示繪圖區(canvas)
  • N12: 新開啟一個 3D 的地圖檢視
  • N13: 新建立一個地圖檢視書籤，可以在書籤檢視中快速跳至該地圖檢視範圍(N14)
  • N14: 顯示地圖檢視書籤
  • N15: 時間管理工具，若圖層有時間的屬性，可設定並顯示連續的動畫
  • N16: 重新整理繪圖區

5.2.1.3 選擇工具列(selection)

  • S1: 選擇圖徵(select features)，可以用下列不同方式選擇
    • 點選(select features)
    • 繪製多邊形來選擇範圍內的圖徵(select features by polygon)
    • 自由繪製範圍來選擇圖徵(select features by freehand)
    • 設定半徑來選擇圖徵(select features by radius)
  • S2: 選擇方式，有以下幾種
    • 由數值來選擇(select features by value)，可依照特定欄位的值，例如選擇縣市名稱
    • 由表示式來選擇(select features by expression)
    • 選擇所有圖徵(select all features)
    • 反向選擇(invert feature selection)
  • S3: 取消選擇，底下有兩個子選項：取消所有圖層中的選擇、只取消選取的圖層
  • S4: 由地點來選擇(select by location)，可從其他圖層和目標圖層的空間關係來選取。 詳細步驟請參見 Chapter 8

5.2.2 面板(panels)

面板是 QGIS 的圖形化介面主要元素，通常是放在在 QGIS 視窗的左方或右方， 裡頭有方便存取的選單、工具列等元件(Figure 5.21)。使用上可以用滑鼠拖拉的方式隨意收放在任一區域， 或是用浮動模式。若使用多個螢幕，也能把面板放在不同的螢幕上，以最大化繪圖區的視窗。

移動的面板的時候，可用滑鼠拖拉至目標位置，此時會出現灰色背景提示(Figure 5.22)。 若要移動為頁籤模式，就將要移動的面板移到既有的面板上(Figure 5.24)，此外頁籤也能夠 用滑鼠拖拉順序。

5.2.3 狀態列

狀態列包含「快速搜尋工具」、「座標或範圍顯示區(coodinates)」、「比例尺(scale)」、 「縮放繪圖區(magnifier)」、「旋轉繪圖區(rotation)」、「座標參考系統」等(Figure 5.24)。

QGIS 的功能非常多，圖形介面也很複雜，通常要執行某個特定地理演算法時，會從選單或工具列來點選。 但若您一時找不到功能在哪裡時，可直接在快速搜尋工具(windows 快速鍵 ctrl + k；mac os 快速鍵 command + k) 中直接輸入您想要找的工具名稱關鍵字即可(Figure 5.25)。舉例來說，如果我要執行向量的環域功能(buffer)， 可在此直接輸入 buffer 或是前幾個字就可以找到。
